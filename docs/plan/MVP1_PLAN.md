# AgentGaia — MVP1 Plan

**목표:** 경영진 시연 → "영업 가능하겠다" 판단  
**기간:** 4주 (주간 시연 사이클)  
**핵심 기능:** 채팅 기반 문서 분석 & Q&A

---

## 1. Executive Summary

### 1.1 MVP1 정의

| 항목 | 내용 |
|------|------|
| **한 줄 정의** | 채팅창에 파일 넣으면 요약하고, 질문하면 찾아서 답해주는 서비스 |
| **타겟 사용자** | 경영진, 직장인, PM (문서 많이 다루는 사람) |
| **킥 포인트** | "ChatGPT는 내 파일 못 읽잖아. 이건 되네." |
| **성공 기준** | 경영진이 "이거 고객한테 보여줘도 되겠다" |

| 컴포넌트 | 상태 | 비고 |
|----------|:----:|------|

## 2. 주차별 계획

### Week 1: "파일 넣으면 답해주네"

**시연 목표:** PPT 업로드 → 요약 → Q&A

**시연 스크립트:**
```
[시연자] 이게 저희가 만든 채팅 서비스입니다.
        여기 30페이지 발표자료 넣어볼게요. (드래그앤드롭)

[채팅창] 파일 업로드 완료. 무엇을 도와드릴까요?

[시연자] "이거 3줄로 요약해줘"

[채팅창] 
1. 3분기 매출 목표는 150억원입니다.
2. 주요 전략은 B2B 신규 고객 확보입니다.
3. 예상 리스크는 원자재 가격 상승입니다.

[시연자] "3분기 목표 달성 방안이 뭐야?"

[채팅창] 
신규 고객 20개사 확보와 기존 고객 업셀링입니다.
(출처: 슬라이드 15)

[경영진] "오, 파일 안에서 찾아서 답해주네"
```

**개발 범위:**
- [ ] 채팅 UI (Streamlit 또는 간단한 웹)
- [ ] 파일 업로드 API
- [ ] PPT 파서 (python-pptx)
- [ ] 임베딩 → Qdrant 저장
- [ ] RAG 검색 → LLM 답변
- [ ] 출처(슬라이드 번호) 표시

**기술 스택:**
- Frontend: Streamlit (빠른 개발) 또는 Next.js
- Backend: FastAPI
- Parser: python-pptx
- Vector: Qdrant (기존)
- Embedding: 기존 서버 연동
- LLM: Claude API

### Week 2: "다른 파일도 되네"

**시연 목표:** Word, Excel, PDF 지원 추가

**시연 스크립트:**
```
[시연자] 지난주는 PPT였고, 이번엔 다른 것도 됩니다.
        이 Word 계약서 넣어볼게요.

[채팅창] 파일 업로드 완료. 무엇을 도와드릴까요?

[시연자] "우리한테 불리한 조항 있어?"

[채팅창] 
제7조 손해배상 조항에서 "갑의 귀책사유로 인한 
손해에 대해 배상 한도 없음"이 불리할 수 있습니다.
(출처: 7페이지)

[시연자] 이번엔 Excel 넣어볼게요.
        "이상한 데이터 있어?"

[채팅창]
3월 15일 매출 데이터가 음수(-500만원)로 
입력되어 있습니다. 확인 필요합니다.
(출처: Sheet1, Row 23)

[경영진] "Word도 되고 Excel도 되네"
```

**개발 범위:**
- [ ] Word 파서 (python-docx)
- [ ] Excel 파서 (openpyxl)
- [ ] PDF 파서 (PyMuPDF)
- [ ] 파일 형식 자동 감지
- [ ] 출처 표시 개선 (페이지/시트/행)

---

### Week 3: "여러 개도 되네"

**시연 목표:** 다중 파일 업로드 & 비교 분석

**시연 스크립트:**
```
[시연자] 이번엔 파일 여러 개 한번에 넣어볼게요.
        (1분기, 2분기, 3분기 보고서 3개 업로드)

[채팅창] 3개 파일 업로드 완료. 무엇을 도와드릴까요?

[시연자] "분기별 매출 추이 정리해줘"

[채팅창]
| 분기 | 매출 | 전분기 대비 |
|------|------|------------|
| 1분기 | 120억 | - |
| 2분기 | 135억 | +12.5% |
| 3분기 | 150억 | +11.1% |

3분기 연속 성장 중이며, 성장률은 소폭 둔화.

[시연자] "3분기에 새로 나온 리스크가 뭐야?"

[채팅창]
3분기에 새로 언급된 리스크:
- 원자재 가격 상승 (1,2분기 없음)
- 환율 변동성 확대 (2분기부터 언급)

[경영진] "여러 문서 비교도 되네"
```

**개발 범위:**
- [ ] 다중 파일 업로드
- [ ] 파일별 컬렉션 관리 (Qdrant namespace)
- [ ] 교차 문서 검색
- [ ] 비교 분석 프롬프트 개선
